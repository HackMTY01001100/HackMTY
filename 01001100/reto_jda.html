Explanation

This machine learning model developed for JDA utilizes multivariable regression to predict future sales. Based on data from previous years.
The program is divided into 7 steps:
1. Filling missing values from the data frame with average sales number from the category
2. Replaces events into binary values to stablish weights for its impact on sales
3. Replaces location with price average per location since there is a correlation
4. Replaces date into day of the week to stablish weight for each by the sales it performs each day
5. Modifies de input csv with the previous steps
6. Obtains the regression coefficient and intercept, then performs the regression for a year (184 entries), thus calculating the sales for that year based on the variables
7. The model then exports the predicted sales into a csv on the root directory
